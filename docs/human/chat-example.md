## 错误类型总结

- **意图含糊**：未明确目标或范围，Agent 需猜测。
- **表达歧义**：同一句可作多种理解，易导致答非所问。
- **逻辑错误**：前后矛盾、条件缺失或因果颠倒。
- **未使用专业术语**：用口语或模糊词替代领域常用术语，影响精确理解。
- **表达不专业**：句式松散、缺少关键信息（如文件路径、操作对象），不利于复现与执行。
