# Documentation

本文档目录包含项目相关文档。

## 📁 目录结构

```
docs/
├── README.md          # 本文档
├── PRD.md             # 产品需求文档（基于当前代码实现）
├── feature/
│   └── i18n.md         # 国际化方案
├── risks/
│   └── path-tag-rename.md # 路径转标签风险说明
└── store/
    └── description.md # Chrome Web Store 上架描述文档
```

## 📚 文档说明

### PRD

- **位置**: `PRD.md`
- **内容**: 页面与功能点梳理、主要/一般功能分级

### Chrome Web Store 上架文档

- **位置**: `store/description.md`
- **内容**: 包含上传到 Chrome Web Store 所需的所有描述内容
  - 简短描述（132 字符限制）
  - 完整描述（英文和中文版本）
  - 使用说明
